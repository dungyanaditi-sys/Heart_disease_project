# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict whether a person is likely to have **heart disease** based on medical attributes using Machine Learning techniques. It helps in early detection and can support healthcare decision-making.

---

## 📂 Dataset Information

* The dataset contains medical information of patients

* Key features include:

  * Age
  * Sex
  * Chest Pain Type
  * Resting Blood Pressure
  * Cholesterol Level
  * Fasting Blood Sugar
  * Resting ECG
  * Maximum Heart Rate
  * Exercise Induced Angina
  * Oldpeak
  * Slope
  * Number of Major Vessels
  * Thalassemia

* Target Variable:

  * **1 → Disease Present**
  * **0 → No Disease**
